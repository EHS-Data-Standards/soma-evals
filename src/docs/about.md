# About

## SOMA Evals

SOMA Evals is an evaluation framework for schema-guided text extraction using the
[SOMA LinkML model](https://github.com/EHS-Data-Standards/soma). 

## Related Projects

- [SOMA](https://github.com/EHS-Data-Standards/soma) -- Schema for Omics and Multi-modal Assays (LinkML)
- [LinkML](https://linkml.io/) -- Linked data Modeling Language
