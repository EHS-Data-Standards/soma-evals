# About

## SOMA Evals

SOMA Evals is an evaluation framework for schema-guided text extraction using the
[SOMA LinkML model](https://github.com/EHS-Data-Standards/soma). It is developed at
[Lawrence Berkeley National Laboratory](https://www.lbl.gov/) as part of the
Environmental Health Sciences data standards effort.
